# Instrucciones para Claude Code

Las reglas de este repositorio son comunes a todos los agentes y viven en `AGENTS.md`.
No hay reglas especificas de Claude Code aparte de las de ese archivo.

Al inicio de cada sesion, antes de responder sobre el proyecto o editar nada, lee
`AGENTS.md` completo y sigue su seccion `Inicio obligatorio de cada sesion`. Ese
protocolo gobierna el arranque, el trabajo y el cierre, e incluye leer el
`CHECKPOINT ACTUAL` de `docs/HANDOFF_TO_NEW_CODEX_CHAT.md`, revisar la coherencia de
la documentacion, comprobar Git y explicarte el punto exacto de reanudacion.

Al terminar, aplica su seccion `Cierre obligatorio de cada sesion`.
